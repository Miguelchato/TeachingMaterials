\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}

\begin{itemize}
\item Datos longitudinales recogen observaciones repetidas de la variable respuesta
a lo largo del tiempo, en un mismo individuo
\item El análisis correcto de estos datos contempla que la correlación entre las medidas de 
cada sujeto es tenida en cuenta
\item A parte de las aproximaciones tradicionales (vistas en la clase anterior), también se puede:
  \begin{itemize}
   \item Utilizar \emph{Ecuaciones de Estimación Generalizadas}: GEE
   \item Modelos lineales mixtos
  \end{itemize}
\end{itemize}

\end{frame}


\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{GEE}
\begin{itemize}
 \item Modelan la esperanza marginal o poblacional incorporando la correlación entre las
observaciones correspondientes a un mismo individuo, y se asume independencia de los individuos
 \item Admiten que la variable respuesta siga una distribución distinta a la Gausiana
 \item Consideran una ecuación de estimación que se escribe en dos partes: una para modelar los
parametros de regresión y la segunda para modelar la correlación
\item son bastante flexibles ya que el modelo sólo necesita explicitar una función "link", una
función de varianza y una estructura de correlación
\end{itemize}

\end{frame}



\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{GEE}
\begin{itemize}
 \item Funcionan bien cuando:
  \begin{itemize}
    \item el número de observaciones por sujeto es pequeño y el número de sujetos es grande
    \item se tratan estdios longitudinales donde las medidas siempre se toman en el mismo instante
          de tiempo para todos los sujetos
  \end{itemize}
 \end{itemize}

\end{frame}


\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{GEE}: Formulación

\begin{enumerate}
 \item Parte sistematica [lo mismo que un GLM]
    $$ g(E(Y_{ij})) = g(\mu_{ij}) = \beta' X_{ij} $$
  donde $i=1, \ldots, n$ y $j=1, \ldots, n_i$, y $n$ denota el número de individuos, y $n_i$ 
el número de medidas repetidas para el individuo $i$-ésimo
 \item Parte aleatoria
    $$ V(Y_{ij})=\nu(\mu_{ij}) \phi $$
 donde $\nu$ es la función de la varianza y $\phi$ el parámetro de escala

 \item Además se tiene que explicitar la estructura de la correlación mediante la 
  \emph{working correlation matrix}, $R(\alpha)$

\end{enumerate}

\end{frame}



\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{GEE}
\begin{itemize}
 \item No es necesaria la especificación de un modelo estadístico. Es decir, no es necesario
conocer $f(y|parámetros)$. Así, son flexibles, pero:
  \begin{itemize}
   \item la estimación de las $\beta$'s no tiene porqué se la mejor posible
   \item la inferencia está basada en resultados asintóticos
   \item los métodos de validación son complicados
  \end{itemize}
 \item La estimación de los parámetros se puede encontrar en muchos sitios (ver por ejemplo
Liang y Zeger, Biometrika, 1986  o Zeger et al, Biometrics, 1988)
 \item si hay datos faltantes (missing) la estimación sólo es correcta si los missing son MCAR (missing
completely at Random)
\end{itemize}

\end{frame}

\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{GEE con R}

Antes de estimar el modelo: 

\begin{itemize}
 \item La función {\tt gee} \textbf{asume} que los datos están ordenados segun el individuo 
 \item La esctructura de correlación puede ser: independence, fixed, stat\_M\_dep, 
    non\_stat\_M\_dep, exchangeable, AR-M and unstructured

 \begin{itemize}
  \item[independence] Es la elección más sencilla e ineficiente, ignorando las medidas repetidas.
   \item[exchangeable] es la también llamada estructura de simetría compuesta o esférica, o
estructura de efectos aleatorios $Cov(X_{il}, Y_{ik})=\alpha$. En este caso todas las correlaciones se suponen iguales:

\item[AR-M] de orden uno (M=1): $Cov(X_{il}, Y_{ik})=\alpha^{|l-k|}$
\item[unestructured] Todas las correlaciones pueden ser diferentes. Adecuada si hay datos suficientes para
estimar todas las varianzas-covarianzas
 \end{itemize}

\end{itemize}

\end{frame}

\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{Modelos lineales mixtos}
Como vimos en la sesión anterior, se podría usar un modelo lineal, pero:
\begin{itemize}
\item Las observaciones repetidas en cada grupo o cluster, no son necesariamente
independientes.
\item Con frecuencia, no solo se quieren tomar decisiones respecto de los grupos o cluster
observados, sino que se quiere valorar el efecto de las variables explicativas en una
población de la que los grupos son una muestra.
\item Puede ser de interés valorar la variación del efecto de x de un grupo a otro.
\item La estimación del efecto medio de las variables explicativas en cada grupo puede ser
muy deficiente si no se recoge la posible variabilidad entre los grupos.
\end{itemize}
\end{frame}

\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{Modelos lineales mixtos}
\begin{itemize}
 \item Modeliza la realación entre la variable dependiente y las covariables
 \item Estima la correlación intra-individuo (se puede especificar una estructura)
 \item Se pueden aplicar a muchas situaciones (datos multinivel, ANOVA, datos longitudinales)
 \item No requieren puntos equidistantes (son covariables - se modeliza el efecto)
 \item Son robustos ante los missing  
\end{itemize}

\end{frame}


\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{Modelos lineales mixtos}


Un modelo mixto se puede representar como:

$$\ y = X \beta + Zu + \epsilon\,\!$$

\bigskip

donde
\begin{itemize}

\item[$y$] son las observaciones, con media $E(y)=X\beta$

\item[$\beta$] es un vector de efectos fijos

\item[$u$] is un vector i.i.d de variables aleatorias con media
 $E(u)=0$ y matriz de varianza-covarianza $\operatorname{var}(u)=G$

\item[$\epsilon$] es un vector de términos i.i.d. correspondientes al error aleatorio con media $E(\epsilon)=0$ y varianza $\operatorname{var}(\epsilon)=R$

\item[$X$ and $Z$] son matrices de regresores que relacionan las observaciones
$y$ con $\beta$ y $u$

\end{itemize}

\end{frame}



\begin{frame}[fragile]{Modelos GEE y modelos lineales mixtos}
\textcolor{OliveGreen}{Modelos lineales mixtos con R}

\begin{itemize}

\item Modelo sencillo para interpretar (modelo lineal mixto con intercept aleatorio)
  
$$y_{ij} = \beta_0 + \beta_1 X_{ij} + a_{ij} + \epsilon_{ij}$$
$$ a_i ~ N(0, \tau_{a}^2) \mbox{ ,} \tau_{a}^2 \ge 0 $$
$$ \epsilon_{ij} ~ N(0, \tau^2) \mbox{ ,} \tau^2 > 0 $$

\item El modelo presenta ahora un intercept aleatorio (centrado en 0) que depende del individuo
$i$-ésimo

\item La varianza del efecto aleatorio recoge la variabilidad entre los diferentes individuos
\item La varianza del error recoge la variabilidad dentro de cada individuo no 
explicada por el modelo. NOTA: si la varianza del efecto aleatorio fuese nula, el modelo
coincidiría con el modelo de efectos fijos o de regresión lineal.

\end{itemize}
\end{frame}
